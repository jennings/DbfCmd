=======
DbfCmd
=======

Overview
=========

DbfCmd is a simple command-line tool to query dBASE IV files.


System Requirements
====================

* Microsoft .NET Framework 2.0


Source
=======

The source code is available at:

    https://github.com/jennings/DbfCmd
